AI-powered chatbot for WordPress with multi-provider support, inquiry detection, and notifications.

== Description ==

AI Chatbot for Business is a powerful WordPress plugin that adds an intelligent chatbot to your website. Connect with multiple AI providers (OpenAI, Kimi, Zhipu AI) and provide 24/7 customer support.

= Key Features: =

* **Multiple AI Providers** - Support for OpenAI (GPT), Kimi (Moonshot), and Zhipu AI
* **Inquiry Detection** - Automatically detect sales inquiries and notify you
* **Dual Notifications** - Get alerts via email and WeChat (ServerChan)
* **Multilingual Support** - Built-in support for English, Chinese, and Arabic
* **Customizable Appearance** - Adjust colors, position, and welcome messages
* **Responsive Design** - Works perfectly on desktop and mobile devices
* **Easy Setup** - Configure in minutes with our intuitive admin panel

= Use Cases: =

* Answer customer questions automatically
* Collect inquiries and notify your sales team
* Provide 24/7 support across time zones
* Qualify leads before human intervention
* Reduce support costs with AI automation

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'AI Chatbot' in your WordPress admin menu
4. Configure your AI provider and API key
5. Test the connection and save your settings

== Frequently Asked Questions ==

= Which AI providers are supported? =

Currently supported providers:
* OpenAI (GPT-3.5, GPT-4)
* Kimi (Moonshot AI)
* Zhipu AI (GLM)

More providers will be added in future updates.

= Do I need to pay for AI services? =

Yes, you need an API key from your chosen AI provider. Each provider has their own pricing:
* OpenAI: Pay-as-you-go based on usage
* Kimi: Competitive token-based pricing
* Zhipu AI: Affordable Chinese AI service

The plugin itself is free to use.

= Is my chat history stored? =

Chat messages are processed in real-time and not permanently stored by default. Inquiry notifications are saved to your WordPress database for record-keeping.

= Can I customize the chatbot appearance? =

Yes! You can customize:
* Widget position (left or right)
* Welcome message
* Input placeholder text
* System prompt (AI behavior)

= How do I get WeChat notifications? =

WeChat notifications are powered by ServerChan (Server酱). Get your free SendKey from https://sct.ftqq.com/ and enter it in the plugin settings.
